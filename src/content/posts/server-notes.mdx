This post type is for recording a deployment from goal to verification: environment, configuration, checks, rollback, and troubleshooting.

<Callout type="warning" title="Security note">
  Do not publish API keys, passwords, SSH private keys, or sensitive server configuration.
</Callout>

## Recommended structure

1. Environment
2. Compose services
3. Nginx configuration
4. Verification commands
5. Rollback method
