这个博客用来记录服务器运维、AI 工具、研究流程和个人项目。第一版是完全静态的：页面在构建阶段生成，服务器上的 Nginx 只负责分发文件。

<Callout type="info" title="设计目标">
  视觉语言尽量接近 Notion 文档或知识库：安静、留白充足、低饱和、以阅读为中心。
</Callout>

## 技术栈

| 模块 | 选择 | 说明 |
| --- | --- | --- |
| 框架 | Astro | 静态优先，构建速度快 |
| 内容 | MDX | Markdown 加组件 |
| 样式 | Tailwind 与 CSS 变量 | 统一亮色和暗色主题 |
| 搜索 | Pagefind | 构建时生成静态索引 |

## 代码块测试

```bash
pnpm install
pnpm build
pnpm audit:build
```
