This blog records server operations, AI tooling, research workflows, and personal projects. The first version is fully static: pages are built ahead of time and Nginx only serves files.

<Callout type="info" title="Design target">
  The visual language should feel like a Notion document or knowledge base: quiet, spacious, low-saturation, and focused on reading.
</Callout>

## Stack

| Module | Choice | Note |
| --- | --- | --- |
| Framework | Astro | Static-first and fast |
| Content | MDX | Markdown plus components |
| Styling | Tailwind plus CSS variables | Shared light and dark themes |
| Search | Pagefind | Static index generated at build time |
